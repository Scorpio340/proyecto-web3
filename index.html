Bienvenido a Academia Web Huancayo

Somos una institucion dedicada a la ensenanza de programacion web para jovenes y adultos de la region Junin. Nuestros cursos estan disenados para que aprendas desde cero, con practica real desde el primer dia.

Aprende HTML, CSS y construye tus propias paginas web. Nuestros docentes tienen experiencia en proyectos reales y te guiaran paso a paso.

Por que aprender programacion web?

La programacion web es una de las habilidades mas demandadas en el mercado laboral actual. Saber crear paginas web te abre puertas a trabajos remotos, freelance y proyectos propios.

En Academia Web te ofrecemos: clases practicas, certificado al finalizar y proyectos reales. Ademas contamos con una comunidad activa de estudiantes que se apoyan mutuamente.
